### Clone the repository

### Install the dependencies

```bash
git clone https://github.com/RenszCamacho/pasapalabra-html
cd pasapalabra-html
```

## Install the dependencies

```bash
#Using npm
npm install

#Using yarn
yarn
```

## Run it live

```bash
#Build dev
npm run dev or yarn dev

#Build for Production
npm run build or yarn build
```

This is deployed on Netlify.
